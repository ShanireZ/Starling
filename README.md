# Starling

车头两侧一对**电子百叶进气口("鳃口")**——固定框 + 多片联动百叶,随车速张合的动态造型件。速度越快,叶片开得越大、前缘下斜露缝(战斗机 / F1 多段开缝的样子)。

**这是一个看得见的视觉 / 动态件,不是性能件。** 不追求下压力,面向 100km/h 以内的日常骑行,主打张合动态本身的炫酷。
