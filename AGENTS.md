# AGENTS.md — Starling

## 项目一句话
车头两侧 C 形电子凤翼,开度随车速 / 气流张合的动态造型件。**视觉件,非性能件。** 详见 [docs/charter.md](docs/charter.md)。

## 语言
所有文档、注释、提交信息一律用中文。

## 设计铁律(任何改动都必须守)
1. **收益封顶** — 价值是视觉炫酷,不是性能。任何"为多一点下压力 / 性能"的复杂度一律砍。
2. **第一周可观察** — 优先做出能看见的最小纵切片(翼面随输入张合),再谈扩展。
3. **单人可造可调** — 不引入需要专家或多人协作才能维护的结构。
4. **禁战线蔓延** — 无 App 实时控制、无云、无 OTA、无多学科集成计划。外部端只做配置 + 数据采集。
5. **外行可读** — 结构与价值要让不懂的人一眼看懂。
