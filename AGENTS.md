# AGENTS.md — Starling

## 项目一句话
**车身侧板(车把下整流罩)**上一对**电子百叶进气口("鳃口")**(固定 U/矩形框 + 多片联动百叶,1 自由度,**开口朝车头迎风**),开度随车速张合——越快开得越大、像**张鳃呼吸**纳入迎面气流的动态造型件;开口通透迎风纳气,有**附带的气流 / 散热收益(温和,非主目标)**。详见 [docs/charter.md](docs/charter.md)。

> 形态沿革:早期「3 片折叠成平面 / C 形凤翼」方案已于 2026-06-25 废弃(承风差/舵机顶不住/易抖振)→ 改百叶式(固定框承风、1 自由度);2026-06-28 定位**车身侧板**、开口由"朝车侧外"改**朝车头迎风**(张鳃呼吸纳气),旧"前缘下斜/侧向羽化/下压力"叙事整套作废。

## 语言

## 设计铁律(任何改动都必须守)
1. **收益封顶** — 主目标是造型 / 动态炫酷;迎风纳气的气流 / 散热是附带温和收益、非目标。任何"为多榨一点性能"而显著增复杂度的改动一律砍。
2. **第一周可观察** — 优先做出能看见的最小纵切片(翼面随输入张合),再谈扩展。
3. **单人可造可调** — 不引入需要专家或多人协作才能维护的结构。
4. **禁战线蔓延** — 无 App 实时控制、无云、无 OTA、无多学科集成计划。外部端只做配置 + 数据采集。
5. **外行可读** — 结构与价值要让不懂的人一眼看懂。
