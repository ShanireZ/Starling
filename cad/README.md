| 料 | 下料 | 用途 |
|---|---|---|
| Ø2 不锈钢轴 | **4 段 × 125mm** | 叶轴(300mm 棒 ×2 根够用) |
| 黄铜管 内2外3 | **8 段 × 6mm** | 轴套,压入框两侧柱孔 |
| M2 螺丝 | 4 颗(≈8mm) | 曲柄销:自攻进曲柄销孔,杆身穿拉杆孔 |
| M2 球头拉杆 | 1 套(杆 ≈25mm 起配) | 舵机臂(8mm 孔位)↔ 拉杆下端孔 |
| M3 自攻 | 2 颗 | 框下双耳 → 台架座立柱 |
| 小扭簧 | 1 只(试配) | 套顶叶轴:一脚插框右柱 Ø2 锚孔,一脚顶曲柄——预紧朝闭合 |

## 装配顺序(台架)

1. 框轴孔压入 8 只黄铜套(紧则用 HOLE_COMP 调大重打框,勿硬敲裂框);
2. 钢轴卡进叶片背脊槽 + CA 胶,居中(两端各露 ≈12.5mm);
3. 叶片连轴从框内穿入两侧轴套;右侧露出端压上曲柄(过盈 + CA),**四只曲柄臂必须同角度**(装配时把叶片全部立到竖直位再粘曲柄);
4. M2 螺丝把四只曲柄销接到拉杆四孔(自攻进曲柄,拉杆孔为松孔,能自由转动);
5. 舵机装支架窗口(自带自攻钉),支架用 2×M3 挂框右柱(竖槽可上下微调);
6. 球头拉杆连舵机臂(取 8mm 孔位)与拉杆下端孔,**用易拉夹/球头螺纹把长度调到:舵机中位 ≈ 叶片 50° 半开**;
7. 顶叶轴套扭簧(预紧朝闭合),断电手感验证:松手全组弹回闭合;
8. 框挂台架座(下双耳 M3 自攻进立柱),整机通电跑 `test` 命令扫角。

## 行程与固件映射

- 机械行程:**闭合 = 10°**(微倾使相邻叶鳞片式搭接,不是 0° 纯竖直——纯竖直相邻叶尖会干涉)→ **全开 = 90°(水平)**;
- 固件把 0-100% 开度映射到舵机脉宽 `us0..us90`:**台架标定** `us0` = 叶片抵到闭合位、`us90` = 水平全开位(`set us0 / set us90` + `save`),CLOSED_DEG 的 10° 偏置被标定自然吸收,固件无需改。

## ★ 到货必须卡尺复测的参数(改完重渲重打)

| 参数 | 现值(估) | 说明 |
|---|---|---|
| `SV_HOLE_SPAN` | 34.0 | 舵机两安装孔中心距(**最关键**) |
| `SV_BODY_L / SV_BODY_W` | 28.6 / 13.2 | 舵机体长宽(窗口配合) |
| `SV_AXIS_OFF` | 9.7 | 输出轴距体端偏移 |
| `BUSH_OD / SHAFT_D` | 3.0 / 2.0 | 黄铜套外径 / 钢轴径 |
| `HOLE_COMP` | 0.2 | 打印孔普遍偏紧 → 调大 0.1 重打;偏松 → 调小 |
